# Quiz Game Web Application

An interactive and user-friendly quiz game built using **HTML, CSS, and JavaScript**, featuring a fascinated modern UI/UX design with smooth animations and score tracking.

---

## Task Overview

The purpose of this project is to create a multiple-choice quiz web application that allows users to answer questions, receive immediate feedback, and view their final score at the end of the quiz. The application focuses on both functionality and an engaging user experience.

---

## Features

- Multiple-choice questions
- Instant answer validation
- Final score calculation
- Restart quiz functionality
- Modern glassmorphism-inspired UI
- Fully responsive design

---

##  Technologies Used

- **HTML5** – Application structure  
- **CSS3** – Styling, animations, and UI design  
- **JavaScript (ES6)** – Quiz logic and interactivity
